# Cloud-Based Task Management App

## Project Description
This project is a cloud-based task management application developed using the Scrum framework.  
The application allows users to create tasks, assign tasks to other users, track task progress, and mark tasks as completed.  
The project simulates a single Scrum sprint to demonstrate Agile teamwork and GitHub collaboration.

## Scrum Sprint Description
This repository demonstrates a simulated Scrum sprint workflow, including:

- Scrum artifacts (Product Backlog, Sprint Backlog, Increment)  
- Sprint planning activities  
- GitHub collaboration using branches, commits, and pull requests  

Each team member contributes through individual feature branches and pull requests to ensure proper Scrum and Git practices.
